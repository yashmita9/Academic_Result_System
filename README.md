# Academic Result Management System

## 📌 Project Overview
Academic Result Management System is a full-stack web application developed to help educational institutions manage student results efficiently.  
The system allows administrators to add, update, delete, and view student marksheets and academic records.

---

## 🚀 Tech Stack
- **Backend:** Java, Spring Boot, REST API
- **Frontend:** Angular, HTML, CSS, Bootstrap
- **Database:** MySQL
- **Tools:** Git, GitHub, Postman, Apache Tomcat

---

## ✨ Features
- Student Result Management
- User Authentication
- Add / Update / Delete Student Records
- View Student Marksheet
- REST API Integration
- Responsive UI
